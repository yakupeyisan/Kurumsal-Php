<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.


### Installation And Start

1. Clone the repo
   ```sh
   git clone https://github.com/yakupeyisan/Kurumsal-Php.git
   ```
2. Install Composer packages
   ```sh
   cd RestApi
   ```
   ```sh
   composer install
   ```
3. Start 
   ```sh
   php spark serve
   ```
